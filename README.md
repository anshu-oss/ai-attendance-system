# 🎓 SnapClass – AI Powered Attendance Management System

SnapClass is an AI-powered attendance management platform that modernizes classroom attendance using **Face Recognition** and **Voice Recognition**. The system provides a seamless experience for both teachers and students with secure biometric authentication, instant enrollment, and real-time attendance tracking.

## ✨ Features

- 🤖 AI-powered attendance system
- 👤 Face Recognition based authentication
- 🎤 Voice Recognition for secure verification
- ⚡ Instant student enrollment via QR Code or Course Link
- 📊 Student attendance dashboard
- 👨‍🏫 Teacher dashboard for attendance management
- ☁️ Cloud-based attendance storage
- 📱 Responsive and modern UI

---

## 🚀 Student Workflow

### Phase 1 – Instant Enrollment
Students join courses instantly using QR codes or invitation links without lengthy registration.

### Phase 2 – Biometric Registration
Students register their Face ID and Voice ID once for secure AI-based authentication.

### Phase 3 – Personal Dashboard
Students can view enrolled subjects, attendance percentage, and attendance history in one place.

## 👨‍🏫 Teacher Workflow
